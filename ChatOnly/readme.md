##劳一劳<br>
####简介<br>
本软件用于附近的人的交友活动，属于定时开放的软件，基于地图查看附近的匹配，特点是为了打造一对一的文字聊天模式，对于出现的地图定位只是示例位置，非真实地理位置，所有地理数据为真实数据，保证聊天对象在附近，除此之外还增加了吐槽和广告等功能，还可以实时与管理员交流。
